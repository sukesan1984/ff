# 🐤 Flappy Fever

攻めるほど気持ちよくなる、フィーバー型フラッピーバード。**Godot 4** 製。
画像素材ゼロ(全部コード描画)・効果音もPythonで手続き生成しているので、依存なしで動きます。

## 遊び方

1. [Godot 4.2 以降](https://godotengine.org/download) をインストール
2. Godot を起動し「インポート」→ このフォルダの `project.godot` を選択
3. ▶(F5)で実行
4. **タップ / クリック / スペース / ↑ / W** で羽ばたく

> CLI なら: `godot4 --path . ` でも起動できます。

## 何が「ただのフラッピー」と違うか

| 要素 | 内容 |
|------|------|
| 🔥 **FEVERモード** | コインやニアミスでゲージを溜めMAXで発動。**無敵・スコア2倍・加速・虹色**の爽快タイム |
| 🪙 **コイン & コンボ** | 連続取得でスコア倍率アップ(最大 x8) |
| ✨ **ニアミスボーナス** | パイプの縁ギリギリ通過で「NICE!」+ボーナス。上手いほど稼げる |
| 🛡️ **シールド** | 1回だけ衝突を肩代わり |
| ⏳ **スローモー** | 数秒だけ世界がスロー。立て直しのチャンス |
| 🧲 **マグネット** | 近くのコインを吸い寄せる |
| 🌅 **時間帯の変化** | 進むほど 朝→昼→夕→夜(星空)→朝 と空・太陽/月・雲・丘がパララックスで流れる |
| 🎚️ **動的難易度** | スコアで速度が上昇、隙間が縮小、上下に揺れるパイプも登場 |
| 🎬 **演出** | 羽パーティクル、画面シェイク、スコアポップ、メダル(BRONZE〜PLATINUM) |
| 🔊 **サウンド** | 10種の効果音をすべて `gen_sounds.py` でコード合成 |
| 💾 **ハイスコア保存** | `user://flappyfever.save` に自動保存 |

## 構成

```
project.godot          # プロジェクト設定(540x960 縦持ち)
Main.tscn              # ルート(Game.gd を1つ付けるだけ。中身は全部コード生成)
icon.svg
gen_sounds.py          # 効果音WAVを生成するスクリプト
sounds/*.wav           # 生成済みの効果音
scripts/
  Game.gd              # 全体制御:状態/生成/当たり判定/スコア/フィーバー/UI/演出/保存
  Bird.gd              # 鳥:物理・傾き・羽ばたき・トレイル・各オーラ
  Pipe.gd              # 上下パイプ(揺れるタイプ対応)
  Coin.gd              # 回転コイン
  PowerUp.gd           # シールド/スローモー/マグネット
  Background.gd        # 時間帯変化のパララックス背景
  Ground.gd            # スクロールする地面
  Hud.gd               # フィーバーバー/パワーアップ表示/メダル
  FloatingText.gd      # 「+1」「NICE!」のポップ
  Sfx.gd               # 効果音マネージャ(プール再生)
```

## 効果音を作り直す

```bash
python3 gen_sounds.py   # sounds/*.wav を再生成
```

## メモ

- 画面内テキストは英語表記です(Godot のデフォルトフォントが日本語/絵文字を含まないため。
  日本語化したい場合は CJK 対応フォントを `Theme` に設定してください)。
- 当たり判定は物理エンジンを使わず、円 vs 矩形の手計算で行っています(挙動が読みやすく軽い)。
