# 🐤 Flappy Fever

一本指で遊べる**ローグライト・フラッピー**。フラッピーバードを起点に、ヴァンサバ／ディアブロ／ローグライクの要素を融合した「攻めるほど強く・気持ちよくなる」ゲーム。**Godot 4** 製、画像素材ゼロ(全コード描画)・効果音もコード生成。

🎮 **遊ぶ(公開中)**: https://flappy-fever-54519886771.asia-northeast1.run.app

## 遊び方
- **タップ / クリック / スペース**で羽ばたく。土管のすき間をくぐってスコアを稼ぐ。
- タイトルの「あそびかた」でルール、「魂の祭壇」で永続強化。

## 技術構成
- **クライアント**: Godot 4.6。`scripts/Game.gd` が中核。画像素材なし(全 `_draw()`)、効果音は `gen_sounds.py` で合成、日本語フォント Mochiy Pop One 同梱。
- **サーバ/公開**: Go 製の軽量サーバ(静的配信 + `/api/scores`)を **Cloud Run**(東京)で配信。ランキングは **Firestore**。非root distroless・CSP等のセキュリティヘッダ・事前gzip配信。
- **デプロイ**: `./deploy.sh <tag>`(コンパイル→スモーク→Web書き出し→コンテナ build/push→Cloud Run deploy を一括検証)。
- **開発フラグ**(env、本番未使用): `FF_PV`=トレーラー録画 / `FF_AUTO`=自動プレイ観察 / `FF_META`=祭壇プレビュー / `FF_BOSS`=ボス早期出現。

## ローカルで動かす
1. [Godot 4.6+](https://godotengine.org) で `project.godot` を開いて ▶。
2. Web書き出し: `godot --headless --path . --export-release "Web" build/web/index.html`
3. サーバ: `cd backend && go run .`(ランキングはADC/Firestore必要。静的のみなら任意のHTTPサーバでも可)
